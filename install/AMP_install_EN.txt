== Installing modules for Alfresco 3.4 ==

1. Stop Alfresco
   a. Run alfresco.sh stop (or alf_stop.bat)


2. Copy AMP files
   a. Copy "recent-comments-alfresco-module-v1.0.amp" into the "amps/" folder of your Alfresco.

   b. Copy "recent-comments-share-module-v1.0.amp" into the "amps-share/" folder of your Alfresco.


3. Install AMP
   a. Run apply_amps.sh (or apply_amps.bat)


4. Start Alfresco
   a. Run alfresco.sh start (or alf_start.bat)
